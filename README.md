# 🎓 Educational Chatbot

An Android-based educational chatbot developed using **Java**, **Android Studio**, **Firebase Authentication**, and **Firebase Realtime Database**. The application helps students quickly access educational content through a rule-based chatbot with predefined responses.

---

## 📖 Project Overview

The Educational Chatbot is designed to provide students with an easy and interactive way to learn fundamental computer science subjects. Instead of browsing multiple websites or books, users can ask educational questions and receive predefined answers stored in Firebase Realtime Database.

This project focuses on providing accurate and consistent responses by using a rule-based approach rather than Artificial Intelligence. It also includes features such as book recommendations, book comparison, user authentication, and chat history management.

## ⚙️ How It Works

1. User registers or logs into the application.
2. Firebase Authentication verifies the user's credentials.
3. The user enters an educational question.
4. The chatbot searches the Firebase Realtime Database.
5. If a matching topic is found, the stored answer is displayed.
6. Chat history is saved for future reference.
7. Users can also browse recommended books and compare different books.
